# Spam Email / SMS Classifier – Project Reflection

## Technologies Used

* Python
* Pandas
* NumPy
* NLTK
* Scikit-learn
* TF-IDF Vectorizer
* Streamlit
* Pickle
* Visual Studio Code (VS Code)

## Project Overview

The Spam Email / SMS Classifier is a Natural Language Processing (NLP) and Machine Learning project developed to classify messages as either Spam or Not Spam. The project includes data preprocessing, feature extraction using TF-IDF, model training, evaluation, and deployment through a Streamlit web application.

## What Worked Well

* Successfully cleaned and preprocessed text data for machine learning.
* Applied TF-IDF vectorization to convert text into numerical features.
* Trained a machine learning model capable of classifying spam and legitimate messages.
* Built a Streamlit web application that provides real-time predictions.
* Successfully integrated the trained model into the user interface.
* Gained practical experience with the complete machine learning workflow.

## What Did Not Work Well

* Some messages with unusual wording or unseen patterns were not classified correctly.
* Model performance depended heavily on the quality and variety of the training dataset.
* Designing a professional-looking user interface required several revisions.
* Faced minor challenges while organizing project files and connecting all components together.

## What I Would Do Differently

* Spend more time comparing different machine learning algorithms before selecting the final model.
* Perform additional testing using more diverse real-world spam messages.
* Focus on improving the user interface earlier in the development process.
* Add more detailed model evaluation and testing before deployment.

## Key Learning Outcomes

* Text preprocessing and cleaning techniques for NLP tasks.
* Feature extraction using TF-IDF Vectorization.
* Machine learning model training and evaluation.
* Model saving and loading using Pickle.
* Building and deploying applications using Streamlit.
* Understanding the end-to-end lifecycle of a machine learning project.
